## 前序已完成能力

- BYOK API Key 模式：
  - `utils/apiKeyStore.ts` 管理 OpenRouter、RunningHub、MiniMax Key 和 OpenRouter 模型名。
  - `components/ApiSettingsPanel.tsx` 支持用户填写、保存和清除本地 Key。
  - 前端不再读取 `VITE_*` 密钥。
- Cloudflare Worker 代理：
  - `src/worker/index.ts` 实现 OpenRouter、RunningHub、MiniMax 白名单代理。
  - `wrangler.jsonc` 配置单 Worker 托管静态资源和 `/api/*`。
  - Worker 不保存平台密钥，只读取本次请求的用户 Key。
- 前端降级流程：
  - 缺 OpenRouter 时使用本地角色文案、立绘 prompt、Live prompt 和语音台词兜底。
  - 缺 RunningHub 时跳过立绘和动态化，显示文字角色卡/占位视觉。
  - 缺 MiniMax 时跳过语音生成，语音槽位显示缺失态。
