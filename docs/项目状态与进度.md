## 当前已知情况

- `npm run build` 会提示 chunk 超过 500 KB，这是当前 Three.js、React 和业务代码集中打包导致的性能警告，不影响构建成功。
- `index.html` 引用了 `/index.css`，构建时提示不存在并保留运行时解析；这是原项目已有行为，目前不阻塞。
- 本地 `.env.local` 里可能仍有旧变量，Wrangler 本地预览会提示读取，但 Worker 代码不使用这些旧变量，且 `.env.local` 未提交。
- `dist/` 作为构建产物保留在本地，用于 Wrangler 本地预览；仓库忽略规则仍忽略 `dist/`。
